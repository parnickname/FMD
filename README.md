# Система управления автосервисом

Курсовая Работа по ФМД Дронова Никиты

Консольное приложение для управления данными автосервиса, написанное на Go с использованием SQLite.

## Описание

Приложение предоставляет функционал для управления:
- Клиентами и их контактными данными
- Автомобилями клиентов
- Мастерами и их специализациями
- Услугами автосервиса
- Заказами на обслуживание
- Аналитикой по работе автосервиса

## Требования

- Go 1.21 или выше
- SQLite3

## Установка и запуск

```bash
cd autoservice
go mod download
go run main.go
```

## Функциональность

1. **Создание и заполнение БД** - инициализация базы данных с тестовыми данными
2. **Просмотр данных** - вывод информации о клиентах, автомобилях, мастерах, услугах и заказах
3. **Аналитика** - статистика по работе автосервиса
4. **Экспорт в JSON** - сохранение всех данных в файл export_data.json
5. **Импорт из JSON** - загрузка данных из файла export_data.json

## Структура проекта

```
autoservice/
├── main.go              # Главный файл с консольным меню
├── models/              # Модели данных
│   └── models.go
├── db/                  # Работа с базой данных
│   ├── connect.go       # Подключение к SQLite
│   └── init.go          # Инициализация и заполнение БД
├── analytics/           # Модуль аналитики
│   └── analytics.go
└── json_io/             # Экспорт/импорт данных
    ├── export.go
    └── import.go
```

## База данных

Проект использует SQLite для хранения данных. База данных создается автоматически при первом запуске.

### Таблицы:
- `clients` - клиенты
- `cars` - автомобили
- `masters` - мастера
- `services` - услуги
- `orders` - заказы
